🎵 ジュークボックス — 使い方
====================================

このアプリは「PCでそのまま使う」ことも「GitHubに上げてスマホで使う」こともできます。

────────────────────────────────────
■ ファイルの説明
────────────────────────────────────
  index.html          … アプリ本体
  音源\               … ここに音楽ファイルを入れる
  音源リストを更新.ps1 … 音源一覧(manifest.json)を作り直すスクリプト
  はじめにお読みください.txt … このファイル

  対応形式: .mp3 .wav .m4a .ogg .flac .aac .opus


────────────────────────────────────
■ PCでそのまま使う場合
────────────────────────────────────
  1. 「音源」フォルダに音楽ファイルを入れる
  2. index.html をダブルクリックで開く（Edge / Chrome推奨）
  3. 「📂 フォルダから読み込む」を押して「音源」フォルダを選ぶ
  4. 曲がボタンになって並ぶ → 押すと再生


────────────────────────────────────
■ GitHubに上げてスマホで使う場合
────────────────────────────────────
  ● 準備：音源一覧を作る
     「音源リストを更新.ps1」を右クリック →「PowerShellで実行」
     （音源\manifest.json ができます。曲を足すたびに実行してください）

  ● アップロード（ブラウザだけでOK・Git不要）
     1. https://github.com/new で新しいリポジトリを作成
        - Repository name: 好きな名前（例: jukebox）
        - Public を選択
        - 「Create repository」
     2. 作成後の画面で「uploading an existing file」をクリック
     3. このフォルダの中身をすべて（index.html / 音源フォルダ / manifest.json など）
        ドラッグ＆ドロップ →「Commit changes」
     4. リポジトリの Settings → Pages を開く
        - Branch を「main」/「(root)」にして Save
     5. 数分待つと公開URLが表示されます
        https://ユーザー名.github.io/リポジトリ名/
     6. そのURLをスマホで開けば、そのまま使えます！

  ● 曲を追加したくなったら
     音源フォルダに足す → ps1を実行 → GitHubに再アップ（変更分をCommit）


────────────────────────────────────
■ 注意
────────────────────────────────────
  ・Public リポジトリなので、URLを知っていれば誰でも音源を聴けます。
    著作権のある市販曲はアップしないでください（自作・フリー素材はOK）。
  ・1ファイル最大100MBまで。
